## 🏗️ 技术架构

### 系统架构
```
前端应用 (Express + JavaScript)
    ↓ HTTP/JSON
后端 API (Go + Gin + GORM)
    ↓ MySQL Protocol
MySQL 数据库
```

### 技术栈

**后端**:
- Go 1.21+ + Gin + GORM
- MySQL 8.0
- JWT + bcrypt
- Swagger

**前端**:
- JavaScript ES6+ + Ant Design CSS
- Webpack 5 + Express
- History API + Fetch API

**开发工具**:
- Git + Docker Compose
- Go testing + testify
- 测试覆盖率 60%
