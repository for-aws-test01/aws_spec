# AWSomeShop Frontend

内部员工福利电商系统 - 前端项目

## 技术栈

- Vanilla JavaScript (ES6+)
- Webpack
- Express
- LessJS (UI Components)
- ESLint

## 项目结构

```
frontend/
├── src/                    # 源代码
│   ├── js/                # JavaScript 文件
│   ├── css/               # 样式文件
│   └── index.html         # HTML 模板
├── dist/                  # 构建产物
├── server.js              # Express 服务器
├── webpack.config.js      # Webpack 配置
└── package.json           # 依赖配置
```
